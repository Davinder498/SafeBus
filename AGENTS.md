# SafeBus Alberta Agent Guide

## Project scope
SafeBus Alberta is a school transportation operations and live bus visibility SaaS foundation. Build clean, scalable milestones for schools, parents, drivers, and administrators.

## Agent permissions
Agents may create and edit repository files required by the assigned milestone, update configuration, and run local checks. Keep changes focused and easy to review.

## Product boundaries
Do not add grade management, report cards, teacher classroom tools, PowerSchool or SchoolEngage replacement features, payments, SMS, Supabase, real GPS, QR scanning, notifications, native mobile apps, or shared packages unless the current task explicitly requests them.

## Scale and security reminder
Design with future scale in mind: 1,000 schools, 100,000 buses, and 1,000,000 users. Protect student privacy and prefer the principle: track the bus, not the child.

## Milestone discipline
Build one milestone at a time. Do not add features outside the task. Before reporting completion, run lint, typecheck, and build, and report any failures honestly.
