# SafeBus Alberta

SafeBus Alberta is a school transportation operations and live bus visibility SaaS foundation for Alberta schools and bus contractors.

## Milestone 1

This repository contains a pnpm/Turborepo monorepo with a React/Vite web app, shared domain types, reusable UI components, placeholder API contracts, and tenant-scoped mock data. It intentionally does not implement real auth, database, GPS, QR scanning, notifications, integrations, payments, SMS, or a native driver app.

## Commands

```bash
pnpm install
pnpm lint
pnpm typecheck
pnpm build
```
